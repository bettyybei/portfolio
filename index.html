Building Bros

E-commerce site for Buildings

Building Bros is a fully functional e-commerce site for the exchange of faux commercial and residential buildings. Selections can be queried from the database with many fields. Users can checkout as guests or sign up to keep track of past orders. Emails are sent when their building deeds have shipped.

Created with Samuel Lam, Barry Weinberg, Richard Chao
Completed in 2 weeks

Technologies Used:
Node.js, AngularJS, Express.js, PostgreSQL, OAuth, Nodemailer, and more

Personally Responsible for:
Working with complex HTML and CSS templates, cart handling, and checkout (w/integration with stripe)

Map of Origins

A data visualization of where our food comes from

Map of Origins was created for an Open Data hackathon run by Global Open Data for Agriculture and Nutrition (GODAN). It is an interactive map of the food world economy that draws lines connecting countries based on the exports of food products. The app makes viewing this particular data set easier and in a more entertaining way.

Other features include:
  • Two interactive charts of the top 5 import countries
  • Use of an open database from FAOSTAT that was narrowed from 400,000+ rows to a more useful database with ~200,000 rows
  • Displays the JSON Object of the query searched
  • A Hamburger Search



    • Created with Pat Sukhum
    Completed in < 16 hours

    Technologies Used:
    D3.js, CanvasJS, jQuery, PostgreSQL, Sequelize, Materialize

    Personally Responsible for:
    Database integration, form submission and querying
